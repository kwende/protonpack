# Decision Log

Record decisions here so the project does not keep re-litigating the same questions.

## 2026-05-03: Build Scope

Decision: Build a personal-use, wearable, classic GB1/GB2-style proton pack and wand for October 2026.

Reasoning: The selected references are strongest for the classic pack, and the timeline needs a clear target.

## 2026-05-03: Reference Priority

Decision: Use Stefan Otto for layout, Sean Bishop for screen-used dimensional sanity checks, and direct build measurements as the final authority.

Reasoning: Stefan gives complete layout coverage. Sean provides screen-used measurement context. The actual pack must still fit the LC-2 frame, chosen materials, and electronics.

## 2026-05-03: Mechanical Approach

Decision: Hybrid construction: aluminum motherboard and metal stress points, 3D printed/finished shell and greeblies, real hoses/loom/labels, serviceable electronics.

Reasoning: This gives realism and durability without turning the pack into an exhausting metal sculpture.

## 2026-05-03: October Target

Decision: Event-ready by 2026-10-01, with October used as buffer and polish.

Reasoning: Working backward from October leaves time for reliability testing and avoids last-week feature chaos.

## Pending: Electronics Path

Decision needed by: 2026-05-24.

Options:

- Kit-backed primary.
- Fully custom primary.
- Hybrid: commercial kit as fallback/reference, custom controller if schedule allows.

Recommended current lean: hybrid. It preserves the fun custom work while protecting the October deadline.
