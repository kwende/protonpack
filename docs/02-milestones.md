# Milestones

Schedule starts on Sunday, 2026-05-03. The target is event-ready by Thursday, 2026-10-01, with October as buffer and polish.

## May: References, Frame, And First Cuts

### Week 1: 2026-05-03 to 2026-05-09

- Cache reference PDFs.
- Print 1:1 key plans.
- Measure LC-2 frame.
- Build cardboard silhouette.
- Decide classic pack variant and target weight.
- Create first GitHub issues from backlog.

Exit criteria: wearable silhouette, verified print scale, first shopping list.

### Week 2: 2026-05-10 to 2026-05-16

- Draft motherboard CAD/template.
- Build temporary motherboard from plywood/MDF.
- Mark battery bay, electronics bay, cable exits, and shell mounting zones.
- Identify long-lead found parts and labels.

Exit criteria: temporary motherboard mounted to LC-2.

### Week 3: 2026-05-17 to 2026-05-23

- Wear-test temporary motherboard.
- Adjust mounting geometry.
- Design final aluminum motherboard.
- Lock electronics direction: kit-backed, custom, or hybrid.

Exit criteria: final motherboard cut file ready and electronics path chosen.

### Week 4: 2026-05-24 to 2026-05-30

- Cut final motherboard.
- Add brackets, rivnuts, standoffs, and strain-relief hard points.
- Start cyclotron mockup.

Exit criteria: metal motherboard on LC-2 and rough cyclotron mounted.

## June: Pack Shape And Wand

### Week 5: 2026-05-31 to 2026-06-06

- Finish cyclotron rough shell.
- Build power cell opening and upper box volumes.
- Start first greeblie prints.

Exit criteria: lower and upper visual mass reads as a proton pack.

### Week 6: 2026-06-07 to 2026-06-13

- Build wand structure.
- Fit handles, switches, and bargraph opening.
- Confirm gun mount and hose/cable route.

Exit criteria: rough wand can mount to pack and accept electronics.

### Week 7: 2026-06-14 to 2026-06-20

- Bench power distribution.
- Bring up power cell and cyclotron LEDs.
- Bring up audio path.

Exit criteria: bench rig can boot, idle, and animate lights.

### Week 8: 2026-06-21 to 2026-06-27

- Wire wand controls to bench rig.
- Add vibration outputs.
- Build first connectorized harness.

Exit criteria: wand controls can trigger pack behavior on the bench.

## July: Alpha Integration And Detail Mass

### Week 9: 2026-06-28 to 2026-07-04

- Install temporary electronics into pack.
- First wearable alpha with lights/sound.
- Log weight, heat, balance, and cable problems.

Exit criteria: rough pack works while worn.

### Weeks 10-11: 2026-07-05 to 2026-07-18

- Refine shell modules.
- Add metal cosmetic plates and bumper parts.
- Finish major printed/found greeblies.
- Improve speaker placement.

Exit criteria: all major exterior parts mounted in rough form.

### Week 12: 2026-07-19 to 2026-07-25

- Second integrated alpha.
- One-hour wear test.
- Create punch list for finish readiness.

Exit criteria: no unresolved structural or power architecture surprises.

## August: Finish Prep, Paint, And Final Electronics

### Weeks 13-14: 2026-07-26 to 2026-08-08

- Sand/fill/prime shell modules.
- Finalize labels and hose/loom cut list.
- Build final electronics mounts.
- Label harnesses and document pinouts.

Exit criteria: ready to disassemble for paint.

### Weeks 15-16: 2026-08-09 to 2026-08-22

- Final electronics assembly.
- Final pack/wand harness.
- Optional smoke/fan integration if base system is reliable.
- Begin paint tests on scrap.

Exit criteria: electronics stable enough to survive disassembly/reassembly.

### Week 17: 2026-08-23 to 2026-08-29

- Final paint and weathering.
- Apply labels.
- Seal where needed.

Exit criteria: no new cosmetic fabrication after 2026-08-30 unless it fixes a failure.

## September: Reliability And Costume Integration

### Weeks 18-19: 2026-08-30 to 2026-09-12

- Full runtime test.
- One-hour and two-hour wear tests.
- Transport test.
- Create event field kit.

Exit criteria: pack can run through realistic event use.

### Weeks 20-21: 2026-09-13 to 2026-09-26

- Repair failures from testing.
- Final costume fit with flight suit, belt, gloves, and boots.
- Final documentation photos.

Exit criteria: event-ready pack.

### Final Buffer: 2026-09-27 to 2026-10-01

- No new features.
- Fix only reliability, comfort, safety, or glaring finish issues.

Exit criteria: wearable, charged, packed, and documented.

## October

October is polish and survival time:

- Repair event wear.
- Improve weathering.
- Replace temporary brackets.
- Tune firmware.
- Build nicer service panels.
- Document lessons learned for version 2.
