# Backlog

Use these as GitHub issues. Keep each issue small enough to finish in one shop session or one evening when possible.

Priority meanings:

- P0: Blocks the October build.
- P1: Strongly affects quality, reliability, or realism.
- P2: Nice polish or version 2.
