# Reference Sources

Use this file to keep the project grounded. Do not let reference gathering become the project; collect enough to build, then measure and fabricate.

## Primary Plan Sources

### Stefan Otto / Bug Stomper Plans

- GBFans page: https://www.gbfans.com/wiki/Stefan_Otto%27s_Prop_Plans
- HProps mirror of individual PDFs: https://www.hprops.com/labels/

Use for: overall CAD-derived layout, module naming, 1:1 print references, motherboard/pack overview, and part placement.

Notes: GBFans describes these as CAD plans made from Planet Hollywood reference photos, with unknown dimensions inferred from known found-object dimensions. They remain very useful, but they are not a final authority for every physical dimension.

### Sean Bishop Plans

- GBFans page: https://www.gbfans.com/wiki/Sean_Bishop%27s_Prop_Plans

Use for: dimensional sanity checks against screen-used pack measurements.

Notes: GBFans says Sean had access to screen-used packs, including a GB1 foam stunt pack and a GB2 fiberglass hero pack. Treat these as strong cross-checks, especially when the Stefan layout looks suspicious or when a found part is known.

## Practical Build Sources

### Studio Creations Proton Pack Tutorial

- Tutorial: https://www.studiocreations.com/HOWTO/gb_protonpack/index.html

Use for: build sequencing, physical reference workflow, motherboard mounting logic, hose/loom cut lengths, and practical assembly lessons.

Important takeaways for this project:

- Print or otherwise keep 1:1 plans physically visible in the shop.
- Reference screen-used packs and museum/display photos more than fan builds.
- Put cable strain loads into the motherboard/frame, not into a thin shell.
- Lights make the pack feel alive even when the shell is imperfect.

### HProps Labels

- Labels page: https://www.hprops.com/labels/

Use for: pack/wand labels, gunbox template, and other print assets. Print at 100 percent and verify scale before applying.

## Electronics Behavior References

### GBFans Power Cell and Cyclotron Light Kit Instructions

- PDF: https://www.gbfans.com/images/shop/packlightinstructions.pdf

Use for: baseline light behavior and architecture ideas. The kit documentation describes a 9V to 14V operating range, a 15-LED power cell, four red cyclotron LEDs, and movie-accurate/special-effect modes.

### GPStar Pack and Wand User Guide

- Manual mirror: https://manuals.plus/gpstar-technologies/proton-pack-and-neutrona-wand-manual.pdf

Use for: feature benchmark if building custom electronics. The manual is useful for thinking through pack/wand interaction, mode selection, vibration, brightness, overheat, smoke behavior, and menu systems.

## Reference Rule

Every physical dimension in this repo should eventually have:

- Source: Stefan, Sean, Studio Creations, direct found-part measurement, or personal measurement.
- Confidence: low, medium, high, or confirmed-on-build.
- Verification: print scale check, caliper measurement, dry fit, or wearable test.
